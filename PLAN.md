# 급식아레나 (Lunch Arena) 기획서

## 한줄 요약
전국 학교급식을 지도에서 탐색하고, 평가하고, 학교끼리 배틀하는 게이미피케이션 웹앱

---

## 1. 컨셉

### 핵심 가치
- **탐색**: 전국 학교 급식을 지도에서 구경
- **참여**: 누구나 별점/사진/글로 평가
- **경쟁**: 학교 간 급식 점수 배틀

### 톤앤매너
- 토스 클린 UI + 아케이드 게임 감성
- 도트풍 학교 아이콘, 배틀 이펙트
- 밝고 따뜻한 컬러 (크림 배경 #fff8f0)
- 학생이 친구한테 공유하고 싶게

### 타겟 사용자
- 1차: 중고등학생 (자기 학교 급식 자랑/까기)
- 2차: 학부모 (학교 급식 품질 확인)
- 3차: 일반인 (재미로 구경)

---

## 2. 학교 지도

### 기술
- 카카오맵 SDK 위에 커스텀 마커
- 학교 위치: 주소 → 카카오 지오코딩 API로 위도/경도 변환
- 지오코딩은 Edge Function에서 배치 처리 후 DB 저장

### 지도 마커 (아케이드 스타일)
- 초등학교: 🌱 초록 원형 마커 (작은 크기)
- 중학교: 📘 파랑 사각 마커 (중간 크기)  
- 고등학교: 🎓 보라 별 마커 (큰 크기)
- 마커 위에 학교 급식 랭크(S/A/B/C/D) 표시

### 지도 인터랙션
- 기본: 전국 줌아웃 → 지역별 학교 클러스터
- 줌인하면 개별 학교 마커 표시
- 마커 클릭 → 바텀시트로 학교 정보 + 오늘 급식
- 내 학교 즐겨찾기 (로컬스토리지)

### 지도 위 정보
- 지역별 평균 급식 점수 히트맵 (선택)
- "오늘의 급식왕" 학교 하이라이트

## 3. 급식 점수 시스템

### 3-1. 자동 점수 (AI 점수, 100점)
시스템이 메뉴를 분석해서 자동 부여. 투명하게 기준 공개.

| 항목 | 배점 | 기준 |
|------|------|------|
| 다양성 | 25점 | 반찬 수 (7개 = 만점) |
| 인기 메뉴 | 25점 | 치킨/떡볶이/피자 등 포함 여부 |
| 영양 균형 | 25점 | 나물/생선/두부 등 건강 식재료 |
| 디저트 | 25점 | 과일/요구르트/케이크 등 후식 |

### 3-2. 유저 점수 (참여 점수)
유저 평가가 쌓이면 학교 종합 점수에 반영.

#### 평가 방법
1. **별점** (필수, 1~5점)
   - 1점: 😫 최악 
   - 2점: 😟 아쉬움
   - 3점: 😐 보통
   - 4점: 🙂 맛있음
   - 5점: 😍 최고

2. **한줄평** (선택, 최대 100자)
   - "오늘 치킨 레전드" 이런 느낌

3. **사진 첨부** (선택, 최대 1장)
   - Supabase Storage에 업로드
   - 자동 리사이즈 (800px)
   - 사진 있는 리뷰는 "인증 리뷰" 뱃지

### 3-3. 종합 점수 (학교 점수)
```
학교 종합 점수 = (자동 점수 × 0.4) + (유저 평균 별점 × 20 × 0.6)
```
- 자동 점수 40% + 유저 점수 60%
- 유저 평가가 없으면 자동 점수 100%
- 유저 평가 10개 이상이면 비율 고정

### 3-4. 랭크 시스템
| 랭크 | 점수 | 칭호 | 색상 |
|------|------|------|------|
| S | 90~100 | 전설의 급식 | 금색 그라디언트 |
| A | 75~89 | 갓급식 | 보라 |
| B | 55~74 | 괜찮은 급식 | 파랑 |
| C | 35~54 | 그저 그런 급식 | 노랑 |
| D | 0~34 | 아쉬운 급식 | 회색 |

### 3-5. 중복 평가 방지
- 로그인 없으므로 브라우저 핑거프린트 기반
- 같은 학교 + 같은 날짜 + 같은 핑거프린트 = 1회만
- 핑거프린트: canvas + timezone + 화면크기 조합 해시

## 4. 학교 간 배틀

### 4-1. 자동 매칭 배틀
매일 시스템이 자동으로 배틀 카드 생성.

#### 매칭 규칙
- **같은 지역 배틀**: 같은 시도 내 학교끼리 (서울 A초 vs 서울 B초)
- **같은 유형 배틀**: 초등끼리, 중학끼리, 고등끼리
- **점수 비슷한 학교끼리**: 점수 차이 ±20 이내 매칭
- 매일 학교당 1개 배틀 자동 생성

#### 배틀 방식
1. **오늘의 급식 배틀**
   - 오늘 메뉴 자동 점수로 승부
   - 유저 투표로 보너스 점수 (+10점)
   - 투표: "어느 학교 급식이 더 맛있어 보여?" 원클릭 투표
   
2. **배틀 결과 카드** (공유용)
   ```
   ┌─────────────────────────────┐
   │     🏆 오늘의 급식 배틀      │
   │                             │
   │  A학교 72점  VS  B학교 68점  │
   │  [메뉴 목록]    [메뉴 목록]   │
   │                             │
   │     🎉 A학교 승리! (+4점)    │
   │                             │
   │  [투표하기]     [공유하기]    │
   └─────────────────────────────┘
   ```

3. **재미 요소**
   - 배틀 시작 시 "3, 2, 1, 배틀!" 카운트다운 애니메이션
   - 승리 학교에 폭죽/컨페티 이펙트
   - 연승 기록: "OO학교 5연승 중! 🔥"
   - 배틀 결과를 카카오톡/인스타 스토리로 공유

### 4-2. 도전장 배틀 (Phase 3)
- 유저가 직접 상대 학교 지정
- "우리 학교가 니네보다 잘 먹어!" 도전장

### 4-3. 시즌제 (Phase 3)
- 월별 시즌 리셋
- 시즌 종료 시 "이 달의 급식왕 학교" 선정
- 학교별 시즌 전적 누적

## 5. 랭킹 시스템

### 5-1. 일일 랭킹
- 오늘 급식 종합 점수 기준 전국 순위
- 지역별 필터 (서울/경기/부산 등)
- 학교 유형 필터 (초/중/고)

### 5-2. 주간/월간 랭킹
- 한 주/한 달 평균 종합 점수
- 꾸준히 좋은 급식 제공하는 학교 상위

### 5-3. 명예의 전당
- 역대 최고 점수 급식 (S랭크 기록)
- 최다 리뷰 학교
- 최장 연승 학교

## 6. 페이지 구조

### 탭 네비게이션 (하단 4개)
1. **🗺 지도** - 전국 학교 지도 + 탐색
2. **🏠 홈** - 내 학교 급식 + 평가
3. **⚔️ 배틀** - 오늘의 배틀 카드 + 투표
4. **🏆 랭킹** - 전국/지역 순위

### 플로우
```
지도에서 학교 탭 → 급식 메뉴 확인 → 별점+사진 평가 → 배틀 투표 → 결과 공유
```

## 7. 기술 스택

| 구분 | 기술 |
|------|------|
| 프론트엔드 | HTML + CSS + Vanilla JS (단일 파일 or 소수 파일) |

| 배포 | GitHub Pages (ssuksak/lunch-arena) |
| API | 나이스 NEIS Open API (학교/급식 데이터) |
